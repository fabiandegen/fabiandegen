# Fabian Degen

MSc student in Advanced Computer Science at the University of Oxford, working on LLM agents and causal reasoning.

- Co-first author of **Plan2Map**, a multimodal benchmark for reconstructing geospatial boundaries from UK planning documents ([arXiv](https://arxiv.org/abs/2606.02747) · [project page](https://odeb1.github.io/Plan2Map_Project_Page/)). I built [GeoPlanAgent](https://github.com/fabiandegen/GeoPlanAgent), the tool-augmented agent system introduced in the paper.
- #4 all-time contributor to [TransformerLens](https://github.com/TransformerLensOrg/TransformerLens), the mechanistic interpretability library. Received a $5,000 Manifund grant to help work on the TransformerLens 3.0 release, which grew the number of supported models from ~200 to ~9,000; my main contributions were the backwards-compatibility layer and weight-level Q/K/V splitting.
